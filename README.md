# made-a-new-game-about-python
我尝试做了一个小游戏,这个是步骤。希望这能给其他新手一点启发。
Guess-Number-Game

一款基于Python编写的经典命令行猜数字小游戏，可打包为Windows可执行文件，无需Python环境即可运行。

项目简介

本项目是入门级Python练手小程序，实现经典猜数字游戏逻辑，操作简单、运行流畅，支持自定义程序图标，可直接分享给他人使用，适合编程新手学习与实操。

游戏规则

1. 程序随机生成1-100之间的整数

2. 用户输入数字进行猜测

3. 程序实时提示数字偏大/偏小

4. 猜对数字后游戏结束，按回车键关闭窗口

文件说明

guess-number-game/
game.py          # 游戏核心源代码
favicon.ico      # 程序自定义图标文件
README.md        # 项目说明文档

运行方式

1. 源码运行

确保本地安装Python环境，在项目文件夹终端执行以下命令：

python game.py

2. 打包为EXE文件（Windows）

安装打包工具

pip install pyinstaller

打包命令（带自定义图标）

pyinstaller -F -i favicon.ico game.py

打包完成后，在dist文件夹中找到game.exe，双击直接运行。

分享说明

1. 打包生成的game.exe为独立可执行文件，无需安装Python

2. 微信/QQ分享时，需将exe文件压缩为ZIP压缩包后发送，避免文件拦截

3. 对方解压后即可直接运行游戏

注意事项

1. 游戏代码已添加防闪退语句，运行结束后不会直接关闭窗口

2. 打包时请勿添加-w参数，否则会导致游戏窗口无法正常显示

3. 自定义图标仅支持.ico格式，替换图标后需重新打包生效
